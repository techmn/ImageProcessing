# ImageProcessing
Basic Image Processing using C#

This application includes following funtionalities
1. Color Operations
    - RGB to Grayscale
    - Negative
    - Sepia
2. Image Filtering
    - Lowpass Filter (Blur)
    - Highpass Filter (Edge Detection)
3. Image Thresholding
4. Color Tint
